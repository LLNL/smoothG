Detailed Description

Hybridization solver for saddle point problems.

This solver is intended to solve saddle point problems of the form

Given , the "element" matrices of and , the following hybridized system is formed

The matrix is the constraint matrix for enforcing the continuity of the "broken" edge space as well as the boundary conditions. This is created inside the class.

Each constraint in turn creates a dual variable (Lagrange multiplier). The construction is done locally in each element.
