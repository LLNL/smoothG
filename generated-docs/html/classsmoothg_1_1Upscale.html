Construct upscaled system and solver for graph Laplacian.

Parameters
graphthe graph on which the graph Laplacian is defined
paramupscaling parameters
partitioningpartitioning of vertices for the first coarsening. If not provided, will call METIS to generate one based on param
edge_boundary_attedge to boundary attribute relation. If not provided, will assume no boundary
ess_attrindicate which boundary attributes to impose essential edge condition. If not provided, will assume no boundary
w_blockthe W matrix in the saddle-point system. If not provided, it will assumed to be zero
