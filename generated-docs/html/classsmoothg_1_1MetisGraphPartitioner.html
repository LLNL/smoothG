Partition a graph with weighted edges in num_partitions.

Parameters
wtablea num_vertexes by num_vertexes wtable representing the connectivity of the graph. table has an entry (i,j) if there is an edge between vertex i and vertex j. The weight of the edge is the value of the matrix
num_partitionsnumber of partitions in which we want to divide the graph
partitioningvector of size num_vertexes. partitioning[v] = p if vertex v belongs to partition p. (OUT).
